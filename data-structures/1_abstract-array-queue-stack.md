Массив - это структура данных, состоящая из последовательных проиндексированных (каждому элементу соответствует индекс) ячеек памяти

Оценка сложности - 1) Статический массив; 2) Динамический массив;
Доступ - О(1), О(1) - за счет индексации
Поиск - О(n), O(n) 
Следующие операции только в Динамическом массиве
Вставка - O(n) - из-за переиндексации массива
Добавление - O(1)
Удаление - O(n) - также из-за переиндексации

Статическому массиву выделяется фиксированный размер памяти, 
Динамический массив при росте вместимости (capacity) может создавать новый массив и переносить данные из прошлого массива в него 

=====================================================================

Очередь (Queue) - это FIFO data structure - Первый зашел, первый вышел
Стэк (Stack) - это LIFO data structure - Последний зашел, первый вышел
